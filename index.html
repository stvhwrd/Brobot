This repository is home to the autonomous robot prototype project developed for ENGR 120 at UVic. Brobot is a prototype for an autonomous nuclear disaster cleanup robot. The predetermined milestones of development guided the evolution of the robot (physically) and its brain (code).

What we used to create and develop

  • VEX Programming Control Starter Kit
  • ROBOTC for VEX Robotics
  • Windows 8.1 (64 bit) via Parallels 10 on Mac OS
  • Set ROBOTC application preferences as per ENGR120 pdf

Milestones

  • Milestone 1 - Concept sketches (Individual) due 4 PM on Monday Feb 16, worth 10%
  • Milestone 2 - Mechanical systems test (Group) due Thursday Feb 26 In lab, worth 25%
  • Milestone 3 - Electrical systems test (Group) due Thursday March 12 In lab, worth 25%
  • Milestone 4 - Software systems test and Integrated operation (Group) due March 26 in lab, worth 40%

Results

Brobot killed it! Our team received an A+ after passing all tests and inspections.
